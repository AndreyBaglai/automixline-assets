**Грунт Wanda 670 HB 4+1 SET (серый) 1 л**  
Wanda 670 HB 4+1 SET – это двухкомпонентный грунт-наполнитель высокой плотности серого цвета, поставляемый в комплекте с отвердителем в соотношении 4:1. Разработан для подготовки поверхностей перед нанесением финишного покрытия. Подходит для стали, оцинкованной стали, алюминия и других материалов, используемых в кузовном ремонте. Обладает высокой наполняющей способностью, что позволяет эффективно выравнивать значительные неровности и дефекты. Продукт характеризуется быстрым высыханием, легкостью шлифования и создает прочную основу для последующих слоев краски. Объем 1 л идеален для профессионального применения в средних по размеру ремонтных работах.

**Основные преимущества:**
- Высокая наполняющая способность
- Удобное соотношение смешивания (4:1)
- Универсальность для различных поверхностей
- Быстрое высыхание и простота шлифования
- Надежная основа для окраски
